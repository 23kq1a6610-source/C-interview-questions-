# C-interview-questions-
1. What is the difference between while and do-while?
Answer:
while checks the condition before executing the loop.
do-while executes the loop at least once before checking the condition.
2. What is an array?
Answer:
An array is a collection of elements of the same data type stored in contiguous memory locations.
3. What is a function?
Answer:
A function is a block of code that performs a specific task.
4. What is recursion?
Answer:
Recursion is a process where a function calls itself until a stopping condition is met.
5. What is the difference between malloc() and calloc()?
Answer:
malloc() allocates memory but does not initialize it.
calloc() allocates memory and initializes all bytes to zero.
6. What is the main() function?
Answer:
main() is the entry point of every C program. Program execution starts from this function
7.What is the difference between break and continue?
Answer:
break exits the loop immediately.
continue skips the current iteration and continues with the next one.
8. What is a structure?
Answer:
A structure is a user-defined data type that groups variables of different data types.
9. What is the difference between = and ==?
Answer:
= is the assignment operator.
== is the equality comparison operator.
10. What is a header file?
Answer:
A header file contains function declarations, macros, and definitions..
